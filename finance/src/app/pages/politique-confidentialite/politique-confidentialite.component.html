Politique de Confidentialité

La protection de vos données personnelles est importante pour nous. Cette politique de confidentialité vous informe sur la manière dont CalculateurFinance collecte, utilise et protège vos données, conformément au Règlement Général sur la Protection des Données (RGPD).

1. Responsable du traitement

Le responsable du traitement des données personnelles collectées sur le site CalculateurFinance est :

Nom du site : CalculateurFinance

Email de contact : calculateurfinance@proton.me

2. Données collectées

2.1 Données que nous NE collectons PAS

CalculateurFinance a fait le choix de minimiser la collecte de données personnelles. Les données que vous saisissez dans nos simulateurs ne sont pas collectées, stockées ou transmises à nos serveurs. Tous les calculs sont effectués localement dans votre navigateur.

2.2 Données collectées automatiquement

Lors de votre navigation, certaines données techniques peuvent être collectées automatiquement :

  • Données de navigation : pages visitées, date et heure de visite, durée de session
  • Données techniques : type de navigateur, système d'exploitation, résolution d'écran
  • Données de connexion : adresse IP (anonymisée)

Ces données sont collectées via des outils d'analyse (Google Analytics) configurés pour respecter la vie privée (anonymisation IP, pas de partage avec des tiers).

2.3 Données de contact

Si vous nous contactez par email, nous collectons votre adresse email et le contenu de votre message uniquement pour répondre à votre demande.

3. Finalités du traitement

Les données collectées sont utilisées pour :

  • Améliorer le site : analyser la fréquentation, identifier les contenus populaires, optimiser l'expérience utilisateur
  • Répondre aux demandes : traiter vos questions et suggestions
  • Assurer la sécurité : prévenir les abus et les comportements frauduleux

4. Base légale du traitement

Le traitement de vos données repose sur :

  • Votre consentement : pour les cookies non essentiels et l'analyse d'audience
  • L'intérêt légitime : pour améliorer nos services et assurer la sécurité du site
  • L'exécution d'un contrat : pour répondre à vos demandes de contact

5. Destinataires des données

Vos données peuvent être accessibles aux tiers suivants :

  • Google Analytics : pour l'analyse d'audience (données anonymisées)
  • Netlify : hébergeur du site (données techniques uniquement)

Nous ne vendons jamais vos données personnelles. Aucun transfert de données vers des pays hors Union Européenne n'est effectué sans garanties appropriées.

6. Durée de conservation

Vos données sont conservées pendant les durées suivantes :

  • Données de navigation : 14 mois (Google Analytics)
  • Emails de contact : 2 ans après le dernier échange
  • Cookies : durée variable selon le type (voir Politique de cookies)

7. Vos droits (RGPD)

Conformément au RGPD, vous disposez des droits suivants concernant vos données personnelles :

Droit d'accès

Obtenir une copie de vos données

Droit de rectification

Corriger vos données inexactes

Droit à l'effacement

Supprimer vos données

Droit d'opposition

Refuser certains traitements

Droit à la portabilité

Récupérer vos données

Droit à la limitation

Geler le traitement

Pour exercer ces droits, contactez-nous à : calculateurfinance@proton.me

Nous répondrons dans un délai d'un mois. Si vous estimez que vos droits ne sont pas respectés, vous pouvez introduire une réclamation auprès de la CNIL.

8. Sécurité des données

Nous mettons en œuvre des mesures techniques et organisationnelles appropriées pour protéger vos données personnelles :

  • Connexion sécurisée HTTPS
  • Hébergement sur des serveurs sécurisés
  • Accès limité aux données
  • Anonymisation des données d'analyse

9. Cookies

Notre site utilise des cookies pour améliorer votre expérience. Pour plus d'informations sur les cookies utilisés et comment les gérer, consultez notre Politique de cookies.

10. Modifications de la politique

Cette politique de confidentialité peut être mise à jour pour refléter les évolutions légales ou les changements dans nos pratiques. La date de dernière mise à jour est indiquée en haut de cette page.

Nous vous encourageons à consulter régulièrement cette page pour rester informé de nos pratiques en matière de protection des données.
